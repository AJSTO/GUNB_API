Welcome to the Building Permissions API

This API allows you to retrieve information about number of building permissions for given unit id divided by category, type of building permission compared to last month, last two months, last three months from given date.

Available units types: voivodeships, countys, cadastral units.

Information based on official website of GUNB (General Office of Building Supervision in Poland): GUNB SITE.

API Endpoints

  • /info - to get all available unit id
  • /aggregates/{unit_id}?date={YYYY-MM} - to get detailed info about given permissions by unit, status for given month, if month not given API will return data for actual month.
